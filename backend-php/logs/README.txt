WhatsApp Reminder System Logs
This directory stores all log files:
  - cron_24h_reminder.log  — Cron job execution logs
  - whatsapp_sent.log      — Successfully sent messages
  - whatsapp_errors.log    — API errors
  - webhook.log            — Incoming webhook events
  - webhook_raw.log        — Raw webhook payloads
  - webhook_status.log     — Delivery status updates
  - db_errors.log          — Database connection errors
  - api_errors.log         — Admin API errors
